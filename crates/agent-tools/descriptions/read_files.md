Batch-read up to 8 UTF-8 files in one tool call. Prefer this over multiple sequential read_file turns when you already know the paths you need. Args: paths (required) — either a string[] for uniform reads OR an object[] like [{path, start?, count?, keyword?}, …] for per-file overrides; start/count/keyword/show_line_numbers apply as defaults when an entry omits them. RF50-A: directory paths auto-expand via gitignore-aware walker (up to 12 files per directory, .gitignore + hidden-file rules honored). Returns { files: [{path, status, content?, error?}], succeeded, failed, expanded_from_dirs? }.
